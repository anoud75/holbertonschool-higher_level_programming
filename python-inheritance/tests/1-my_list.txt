===========================
How to Use 1-my_list.py
===========================

This module defines a class ``MyList`` that inherits from ``list``.

Usage
=====

``MyList`` works like a regular list but has a ``print_sorted()`` method.

::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

::

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

::

    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:

::

    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Already sorted list:

::

    >>> sorted_list = MyList()
    >>> sorted_list.append(1)
    >>> sorted_list.append(2)
    >>> sorted_list.append(3)
    >>> sorted_list.print_sorted()
    [1, 2, 3]

List with negative numbers:

::

    >>> neg_list = MyList()
    >>> neg_list.append(-1)
    >>> neg_list.append(5)
    >>> neg_list.append(-3)
    >>> neg_list.print_sorted()
    [-3, -1, 5]

Single element:

::

    >>> single = MyList()
    >>> single.append(42)
    >>> single.print_sorted()
    [42]
