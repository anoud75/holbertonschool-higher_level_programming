===========================
How to Use 4-print_square.py
===========================

This module defines a function ``print_square(size)``.

Usage
=====

``print_square(...)`` prints a square of # characters.

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

::

    >>> print_square(1)
    #

::

    >>> print_square(0)

::

    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
